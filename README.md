# Task Manager CLI Application

## Project Description:
This is a simple command-line interface (CLI) application to manage tasks. The application allows users to:
- Add new tasks
- View existing tasks
- Delete tasks
- Mark tasks as completed
- Save and load tasks from a file (`tasks.json`).
